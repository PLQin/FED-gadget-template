说明文档

基于 jQuery UI 开发的信息提示组件，以弹出层显示提示信息

使用方法与参数说明： 

$( "selector" ).tips({
	content: "",		//提示信息文本
	position: "top", 	// valid value: "top", "bottom", "left", "right"
	maxWidth: "200px",	//弹出层最大宽度
	delay: 3000,		//自动消失时间
	fadeIn: 200,		//淡入时间	
	fadeOut: 200,		//淡出时间		
	color: "red", 		//弹出层背景颜色，可选参数： "green", "red", "blue", "black"
});
