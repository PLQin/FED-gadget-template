使用方法与参数说明： 

$( "selector" ).lightbox({
    width: "auto",      //弹出层的宽度，可以是number或者"auto"
    height: "auto",     //弹出层的高度，可以是number或者"auto"
    autoShow: true,     //初始化之后是否自动显示弹出层
    buttons: [{         //弹出层的操作按钮配置
        type: "button",
        text: "按钮",
        handler: function() {
            $.lightbox( "hide" );
        }
    }, {
        type: "text",
        text: "文本"
    }, {
        type: "link",
        text: "链接",
        handler: function() {
            $.lightbox( "hide" );
        }
    }]
});

可以程序控制弹出层何时显示或者隐藏： 
$.lightbox( "show" );
$.lightbox( "hide" );
